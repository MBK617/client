module:Utils/Redux.injectSaga (key, saga, mode)function static

Dynamically injects a saga, passes component's props as saga arguments
Name Type Description
key string A key of the saga
saga function A root saga that will be injected
mode string optional By default (constants.DAEMON) the saga will be started on component mount and never canceled or started again. Another two options: - constants.RESTART_ON_REMOUNT — the saga will be started on component mount and cancelled with `task.cancel()` on component unmount for improved performance, - constants.ONCE_TILL_UNMOUNT — behaves like 'RESTART_ON_REMOUNT' but never runs it again.
Returns:
